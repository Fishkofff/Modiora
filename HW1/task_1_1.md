### 1. Отличие сущностей JVM, JRE и JDK
Отличие заключается в их предназначении.
* **JVM** нужна для кроссплатформенности (выполняет байт-код) 
* **JRE** объединяет в себе возможности JVM + набор библиотек. 
* **JDK** объединяет в себе все из JRE + javac, jdb, javadoc и тд - инструменты для разработчика. 
**Вывод:** Конечному пользователю достаточно иметь JRE на своем устройстве для выполнения написанного кода, разработчику - JDK.

### 2. Байт-код и "Write Once, Run Anywhere"
**Байт-код** - промежуточный код, который может прочитать JVM. Находится между кодом программиста и машинным кодом процессора. **WORA** - написали код и скомпилировали в байт-код один раз, чтобы исполнить его на разных ОС.
